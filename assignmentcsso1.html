  2. Internal CSS: The second div uses internal CSS, where styles are defined inside the style tag within the head section of the html document.

    4. When you target the same element with inline, internal and external css the styles will be applied in the following order of priority from highest to lowest

  1. Inline CSS: Styles applied directly to the htnl element using the styles attribute . This method has the highest priority.

  2. Internal CSS:Styles defined within the style tag in the head section of the html document . These styles will override external styles but will be overridden by inline styles.

  3. External CSS: Styles defined in an external stylesheet . These styels have the lowest prioprity and can be overridden by both inline and internal styles.

Please note that the external CSS part is commented out in this example . if you create a seperate styles.css file and link it in the html , the external styles will take effect.
